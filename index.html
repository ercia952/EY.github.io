Interactive Installation Processing LeapMotion

Breathink

A breath-visualisation installation that merges breathwork therapy and art therapy — using hand gestures to guide users into calmer breathing rhythms.

Tech Stack

Processing (Java) + LeapMotion sensor + projection mapping

Design Concept

Breathing bubble changes size & position following hand gestures — up to inhale, down to exhale

Outcome

Users unconsciously slowed their breathing when interacting; positive feedback on emotional calm

Interactive Design Arduino Children's UX

Cute Pet Hearing Aid

Animal-shaped hearing aids with an interactive companion app — encouraging hearing-impaired children to wear their devices through play and positive reinforcement.

Target Users

Children aged 6–10 with hearing impairments and their parents

Prototype

3D-printed puppy hearing aid + Arduino ultrasonic sensor detecting wear state + Processing app

Game Loop

Wear aid → puppy walks → collect sounds → earn snack rewards → unlock new animals

System Design IoT Smart City

Ezy Park

An intelligent automatic parking system addressing China's severe urban parking shortage — from mobile reservation to autonomous vehicle retrieval.

Problem

China's parking ratio sits at 1:0.5 in mid-sized cities; 430M+ registered vehicles

System

App reservation → licence plate scan → autonomous parking → real-time monitoring → scan-to-exit

Tech Experiment

Microcontroller-based obstacle avoidance & precision positioning using ultrasonic sensors
